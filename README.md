# photoexplorer

A local disk(or nas) photo(or video) browser (web page in browser), simple and easy to use

搭建一个媒体服务器: 写一个mac app, 放在nas里的photo文件夹, 打开后, 生成全部图片的缩略图, 然后弹出浏览器, 在浏览器里浏览缩略图, 然后如果是打开mov的视频, 则在finder里打开, 其他直接在浏览器里打开.
